# Manual de usuario - ConectaVacantes

## Objetivo
ConectaVacantes ayuda a encontrar vacantes remotas alineadas con un perfil técnico, preparar mensajes de postulación y hacer seguimiento del proceso.

## Flujo de uso
1. Inicia sesión con correo y contraseña o Google.
2. Sube tu CV.
3. Escribe un puesto o habilidad de interés.
4. Ajusta filtros como región, idiomas y tipo de contrato.
5. Revisa las vacantes recomendadas.
6. Genera una propuesta o carta de presentación.
7. Mantén el seguimiento desde el panel de procesos.

## Recomendaciones
- Usa palabras clave específicas del rol.
- Prueba con filtros regionales para encontrar más opciones remotas.
- Revisa el panel de analytics para ver el avance.

## Soporte básico
- Si la API no responde, el sistema usa resultados locales de respaldo.
- Si el login falla, revisa el correo y la contraseña o intenta Google Sign-In.
