# Color Cloud Script ☁️✨

Willkommen bei **Color Cloud Script (CCS)**! Dies ist die offizielle Erweiterung für die einfachste deutschsprachige Programmiersprache der Welt.

## Features
- **Deutschsprachig**: Programmiere mit Befehlen wie `Schreibe`, `Wenn` und `Variable`.
- **Einfachheit**: Keine komplizierten Klammern (außer du willst sie!).
- **Schnelligkeit**: Sofortige Ausführung mit dem CCS-Interpreter.

## Syntax-Beispiel
```ccs
-- Das ist ein Kommentar
Variable name = "Cloud-Entwickler"
Schreibe "Hallo " + name

Wenn 10 > 5:
    Schreibe "Logik funktioniert!"
Ende
```

Entwickelt für die **Color Cloud Community**.
